## 蒸馏方法论

### 1. 数据收集

- **总样本**：363 个斌哥世界视频文本转录
- **时间跨度**：第1集至第132集以上
- **覆盖范围**：恋爱、人生经验、赚钱思维、心理学等多个主题领域

### 2. 样本选择方法

采用系统性分层采样确保代表性：

- **有标题文件**：完整纳入所有32个有标题集数（第1-31集及100-107集等）
- **无标题文件**：采用均匀分布策略，从331个无标题文件中每7个选1个，共约47个
- **采样目标**：总计约100个样本，覆盖整个内容库的时间和主题维度

### 3. 提取维度

从每个视频文本转录中提取以下关键维度：

- **心智模型**：在5次以上不同视频中重复出现的底层思想框架
- **决策启发式**：可直接应用于实际情景的、具有行动导向的建议和方法论
- **表达 DNA**：斌哥的标志性表述方式、比喻、论述逻辑
- **反模式和陷阱**：频繁被提及的错误行为模式和应该避免的做法
- **道德边界**：明确的价值观立场和伦理原则

### 4. 质量控制（三重验证）

所有纳入 Skill 的观点和模型都需要通过以下三重验证：

- **跨领域一致性验证**：在多个不同应用场景中是否逻辑自洽，是否支持一致的世界观
- **预测力验证**：这个观点是否能准确预测现实中的人际互动结果，是否经过验证
- **排他性验证**：这个概念是否具有独特的、不可被其他观点替代的价值

只有同时通过三重验证的观点才被纳入最终的 Skill 输出。
